# @csvshape/core

Core CSVShape utilities for dialect sniffing, encoding detection, Miller-style verb execution, JSONL query shaping, replay-state encoding, and export serialization.

## Install

```bash
npm install @csvshape/core
```

## What it exposes

- Input inspection and dialect detection
- Browser-side verb-chain execution helpers
- JSONL query shaping helpers
- Replay URL encode/decode helpers
- CSV, TSV, NDJSON, and Parquet-adjacent export serialization helpers
